It extracts the word of a given index in a text.

## Usage

```javascript
$ npm install index-to-word
index-to-word@1.0.0 node_modules/index-to-word
$ node
> var indexToWord = require('index-to-word')
undefined
> indexToWord('here we go', 6)
'we'
```
